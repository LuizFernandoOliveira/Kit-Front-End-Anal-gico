# Kit Front-End Analógico

![Status do Projeto](https://img.shields.io/badge/Status-Desenvolvimento-yellow)
![PCB](https://img.shields.io/badge/PCB-KiCad-green)

## Descrição
Kit didático completo de Front-End para circuitos analógicos. Este kit é perfeito para se ter na bancada, pois possui diversas funcionalidades úteis durante o desenvolvimente de sistemas microcontrolados e IoT.

## Descrições técnicas

- Possui LDOs de 3V3 e 5V;
- Conectores USB tipo-C e Jack para alimentação do circuito;
- Possui AmpOp Full Diferencial;
- Possui ADC Full Diferencial:
- Possui Filtro ativo de 2ª Oedem Passa-altas, Passa-baixas, Passa-faixas, Rejeita-faixas;
- Possui Filtros passivos;
- Possui Buffer;
- Possui AmpOp de instrumentação;
- Possui divisores de tensão;
- Possui resistores de Pull-Up e/ou Pull-Down;
- Possui AmpOp RRIO, Ultra-Low Power, Ultra-Low Noise Não Inversor, Inversor, Integrador, Diferenciador;
- Possui chaves analogicas;
- Possui chaves digitais;
- Possui botões e LEDS;
- Possui sensores analógicos;
- Possui diodos Schotky;
- Possui diversas tensões de referência;
- Possui diversos charge pumps.

---

## Estrutura do repositório
Os arquivos de hardware foram desenvolvidos utilizando o KiCad 9.0.
- **`.kicad_pcb`**: Layout da placa de circuito impresso (Board).
- **`.kicad_sch`**: Esquemático eletrônico do sistema (Schematic).
- **`.kicad_pro`**: Gerenciador de projeto do KiCad (Project).
